About The Project

 

 

 

Idea

 

This idea which came out of sheer laziness makes the task easy to the users who frequently use different search engines. Basically, the users who are dissatisfied by the results obtained from one search engine switch to other engines thinking of obtaining better results. Here, this idea is mainly emphasized on and is implemented in such a way the user can avoid searching a single query in several places as he/she can view them in a single window. A quest for finding a most relevant search not only as an article or definition, but want to know the community talk on particular query and at the same time what’s the present thing happening over that query as subject.

Implementation

 

So, in order to implement this, a customized search engine rather to say A Search Aggregator is needed which made us form the MyCollect! Here a user can have the following results: -

  • Overture
  • Social Collect
  • Web Collect
  • News Hotspot

 

         Overture: This gives an overview of the searched query with

1.      Wiki description: A description about query.

2.      Yahoo Images    : User can view top 10 image results about the query posted.

3.      Yahoo News      : lists the top 10 results on the current events about the query posted.

         Social Collect: This is a place where user gets the community talk i.e. what the people around the world are speaking or feeling about the query and the popular bookmarks regarding that query. The images uploaded by several online users around the world can also be viewed here. These are accomplished through following-

1.      Tweets from Twitter.

2.      Bookmarks from Delicious.

3.      Images from Flickr.

         Web Collect: This is the place where user can make his mundane search, which reduces the tediousness in searching the same query in different search engines. Here user gets results from different search engines aggregated neatly in 3 columns without the need of opening in different browser windows. Here we find the search results from-

1.      Google Search.

2.      Yahoo Search.

3.      Bing Search.

         News HotSpot: This is the place where we made the actual intelligence come into picture through News Analysis. The query is analyzed and presented to the user as spots on the yahoo maps (referred to HotSpot) where the spots tell us about the things which are currently spoken about or more happening about the posted query. And also the description of the news is displayed as you hover over the hotspot marked.
